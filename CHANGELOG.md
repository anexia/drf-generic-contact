# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0]

### Added
- Support for Django 5.2 and 6.0
- Support for Python 3.14

### Removed
- Support for Django 5.0 and 5.1
- Support for Python 3.9

### Changed
- Migrate packaging/configuration to `pyproject.toml` and remove legacy setup files
- Install build extras in CI workflows for tests and publishing

## [1.1.0]

### Added
- Support for Django 5.0 and 5.1
- Support for Python 3.12 and 3.13

### Removed
- Support for Django 3.2 and 4.1
- Support for Python 3.8

## [1.0.1]

### Updated
- install_requires in setup.py
- README.md

## [1.0.0]

### Added
- Initial setup
